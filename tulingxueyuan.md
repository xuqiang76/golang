###基本语法(以后展开)###

- 常量 变量 选择 循环 指针 数组 容器
- 面向接口 结构体 组合的概念等
- go语法特色
    - 类型声明 - 放在后面 ... 优点...
    - 格式 - { } 放置位置
    - 格式 - 大小写字母区分对外开放权限
    - 格式 - :=
    - 限制功能 - 无隐式类型转换
    - 限制功能 - import不能导入没使用的包
    - 限制功能 - 不能对指针进行运算
    - 注意 - defer  何时执行，何时不会执行
    - 注意 - select 何时阻塞，何时不阻塞
    - 注意 - break，continue 后面指定标签
    - 注意 - slice 切片作为参数传递时
    - 注意 - byte  []byte,bytes包,高效使用字符串
